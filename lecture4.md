# Angular Component 생성하기

> src / app / app.module.ts

## @ : Decorator
- @NgModule : 해당 클래스틑 모듈로 만들어주겠다는 명시
    - 모듈 : 독립 가능한 기능단위의 묶음
- @Component : 해당 클래스를 컴포넌트로 만들어주겠다는 명시
    - selector : tag name
    - templateUrl : 가져올 html
    - styleUrls : 가져올 css
